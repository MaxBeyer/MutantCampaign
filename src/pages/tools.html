Reactive Effects

Some mutations have what’s called reactive effects. These are marked (R) on the Mutation Cards and in this chapter. In a conflict, you can activate a reactive effect any time you want, and any number of times in a single turn. Activating a reactive effect does not count as an action or as a maneuver – all you need is MP to spend.

Enhancing Effects

Another type of mutation effect is the enhancing effect. These are marked (E). You activate an enhancing effect at the same time as you perform an action or a maneuver, to enhance it. Using an enhancing effect does not count as an action or a maneuver in itself. When using an enhancing effect with a skill roll, roll for the mutation first and then for the skill.
